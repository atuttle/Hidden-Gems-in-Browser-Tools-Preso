Hidden Gems » Console


Pre-req: Everyone knows about console.log, right?


console.log(foo);

May you never alert() again!

Console » Everywhere!


In any tab other than the console,
hit Esc to open a mini-console.

Console » Labels!

console.log() takes multiple arguments.
Label your debug output!


console.log('foo:', foo);

//also handy!
console.info('info:', foo);
console.warn('warn:', foo);
console.error('error:', foo);
console.assert(false, 'my assertion');

Console » KB Shortcuts


⬆ + ⬇ arrow keys scroll through command history


Console » Settings


Persist contents after redirect


  • Right-click in console
  • Button in console toolbar

Console » Dealing w/ Frames


  • Dropdown
  • Change console context:
cd(window.frames['frameName']);

Console » Copy to Clipboard


Copy the result of a console command to the clipboard:


copy('I\'m in ur clipboard, pastin ur text!');

Console » Math


Just type in your math problem and get the answer:


Math.sqrt(1787569);

Hidden Gems » Scripts

Scripts » Step Debugger

  • Breakpoints

  • Break on JS errors

  • Break on DOM change

Scripts » Un-minify

  • Chrome & Safari: { } icon
  • IE9: Tools icon > Format JavaScript

Scripts » "Watch" variables

Auto-updating value list for specified variables

Hidden Gems » Elements/ HTML/ Documents


... because apparently we can't even agree on a standard for tab names.

HTML » Element Dimensions


  • Select the element to see its dimensions, or use the magnifying glass icon in bottom toolbar
  • Layout tab or list of computed styles

HTML » Expand HTML

  • * key on numpad expands everything but scripts & stylesheets. Add shift to include those.
  • Button on documents tab

HTML » Incrementing Numeric Property Values

(All) Up/Down change by 1

Chrome, Firefox, Safari: Shift 10

Chrome, Safari: Alt 0.1

Chrome, Safari: Shift+PgUp/PgDn 100


Bonus: Chrome, Safari, Opera: Also works on colors

HTML » Element State

Preview focused, hover, active, visited states

HTML » Color definition mode

Chrome, Safari: Clicking the color swatch toggles between hex, rgb, hsl, named, etc.

Opera: This brings up a color selection dialog

Hidden Gems » Resources


Lists stylesheets, scripts, images, etc used on the page

Firefox & IE don't have a resources tab but some functionality is implemented in other tabs

Resources » Save changes

Chrome: right click, save as

Safari: Copy + Paste

Resources » Cookies & Storage

Chrome: Resources tab also shows local storage, cookies, caches, etc., and data stored in them

Opera: Storage tab does same thing

Hidden Gems » Network


Usually needs to be open to work, you may have to refresh

Network » Disable cache

Chrome: Settings cog

Firefox: Tab menu

IE9: under "cache" in menu bar

Safari: Develop menu > disable cache

Opera: can't disable, can only clear


All: Ctrl/Cmd + Shift + Delete: Clear cache & other data dialog

Network » Spoof User-Agent

Chrome: settings cog (Just graduated from dev channel to stable in February)

Firefox: User-Agent Switcher extension

Network » Latency

Displays are all slightly different but they all show wait time, download time, etc for each external resource

Network » Events

  • DomContentLoaded
  • Load (images, subframes, etc)

Chrome, Safari

Oh noes, my tools crashed!


Often, using the keyboard shortcut to toggle them
off & on will fix it
