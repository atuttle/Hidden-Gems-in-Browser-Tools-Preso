Hidden Gems in Browser Tools


The part that you've all been waiting for...

Hidden Gems » Console


Pre-req: Everyone knows about console.log, right?


console.log(foo);

May you never alert() again!

Console »
Console Everywhere!


In any dev-tools tab other than the console,
hit Esc to open a mini-console.

Console »
Log Labels!

console.log() takes multiple arguments.
Label your debug output!

console.log('foo:', foo);
//also handy!
console.info('info:', foo);
console.warn('warn:', foo);
console.error('error:', foo);
console.assert(false, 'my assertion');

Console »
Simple Timing

console.time('myTimer');
for (var i=0; i<100; i++){ console.log(i); }
console.timeEnd('myTimer');

Console »
Bling Bling!

Similar to jQuery DOM selection, but without jQuery.

$$('li');

Console »
Inspect

Jump to DOM node in Elements panel from console

inspect( $$('li')[0] );

Console »
KB Shortcuts


⬆ + ⬇ arrow keys scroll through command history



Console »
Settings


Persist contents after redirect


  • Right-click in console ⎚
  • Button in console toolbar ⎚

Console »
Dealing w/ Frames


    The console can only interact with one page. If you've got a frameset or iframe to deal with, it can feel troublesome.


  • Dropdown ⎚
  • Change console context:
cd(window.frames['frameName']);

Console »
Copy to Clipboard


Copy the result of a console command to the clipboard:


copy("I'm in ur clipboard, copyin ur text!");

Console »
Math


Just type in your math problem and get the answer:


Math.sqrt(1787569);

Hidden Gems » Elements / HTML / Documents

HTML »
Element Dimensions


  • Select the element to see its dimensions, or use the magnifying glass icon in bottom toolbar

  • Layout tab or list of computed styles

HTML »
Expand HTML

  •  *  on numpad expands everything but inline scripts & stylesheets. Add shift to include those.

  • Button on documents tab

HTML »
Real Time Editing

Add, Edit, & Delete HTML & CSS inside the browser

HTML »
Incrementing Numeric Prop Vals

  • ⬆/⬇ change by 1
  • Shift+⬆/⬇ change by 10
  • Alt+⬆/⬇ change by 0.1
  • Shift+PgUp/PgDn change by 100

  • Also works on colors (Bonus!)

HTML »

This brings up a color selection dialog
